2.4. EVT: Hydro Events¶

The Hydro EVT is used to setup the flow parameters related to the water-borne hazard event. This can either be a coupling between the shallow-water and CFD solver or using a CFD solver alone. The steps of the setup process for the event has been kept nearly similar to that often used in Finite Element Method to make it easier for researchers from structural engineering background to easily use the app. This includes project settings >> geometry >> meshing >> materials >> initial conditions >> boundary conditions >> solver settings. These steps are available through a tree-structure as shown in EVTSettings.

The steps involved in setup of the Hydro event¶

Each of the above steps in the setup process differs depending on the type of simulation considered. They will be outlined in detail below:

  • Project settings
  • Geometry
  • Meshing
  • Materials
  • Initial conditions
  • Boundary conditions
  • Solver
