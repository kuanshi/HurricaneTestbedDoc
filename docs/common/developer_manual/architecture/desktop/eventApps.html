2.2.2.3. Event ApplicationsÂ¶

The event application takes the event(s) selected for a building site, associates each one with a simulation, and creates EVENT files containing the event intensity measures. It obtains the name of the event from the BIM file and the intensity measure information from the input event files. This input information is saved in an âEVENT.jsonâ file, located in its corresponding simulation working directory.
